# TOAHEX

**TOAHEX** — 3DS Tales of the Abyss Save Editor

本程序基于 C# 语言进行编写，用于 3DS《深渊传说》(Tales of the Abyss) 的游戏存档编辑器，兼容日版、欧版、英版。
程序所读取存档文件必须是未经主机唯一密钥加密，因此请使用存档管理软件(如 Checkpoint、 JKSM)以从主机中导入导出存档 .
这个程序基于对Code.bin的静态分析进行制作，仿照游戏读取文本和对应的编号，用于汉化测试

## 支持的存档类型
| 文件类型 | 文件大小 | 说明 |
|---------|---------|------|

## 功能特性
### 全局数据编辑
- 金币 (Gald) 修改
- 游戏时间修改
- 遇敌次数 / 命中次数修改
- 难度切换
- Grade 修改
- 队伍编排 & 领队设置
- 功能解锁 (C·コア / FSチャンバー)
- 日志全开 / 道具图鉴全开

### 角色编辑
- **基础属性**: 等级、经验值、HP/TP、成长点数、当前称号
- **战斗属性**: 物攻/物防/谱攻/谱防/敏捷/幸运、OVL量表
- **C-Core 加成**: 各项 C-Core 属性加成编辑
- **装备编辑**: 武器、防具、饰品×2、响律符
- **术技编辑**: 快捷术技设置、已学习术技开关、术技使用次数
- **AD 技能**: 88 种 AD 技能开关
- **称号编辑**: 称号全开 / 单个称号解锁

### 道具管理
- 全部 640 种道具数量编辑 (0~99)
- 按类别筛选 (素材/グミ/瓶/强化道具/武器/防具/装饰品等)
- 一键获得全部道具 / 按类别获得
- 一键获得全部装备

### 料理修改
- 20 种料理习得开关
- 各角色料理熟练度编辑 (0~255)

### FSチャンバー (谱石管理)
- 各角色术技的谱石装备类型编辑
- 四色谱石 (赤/青/緑/黄) 等级编辑 (Lv.1~6)
- 谱石最大值设置
- 一键全满 / 一键重置
### 系统数据 (TOASYS)
- 难度模式、金币、游戏时间
- 总游戏时间、存档计数
- 系统标志、遭遇数
- 角色使用次数
### 其他功能
- 存档校验和验证
- 保存时自动备份 (.bak)
- 拖拽打开存档文件
- 键盘快捷键 (Ctrl+O 打开, Ctrl+S 保存)
- 中文/日文界面切换
## 使用方法
1. 启动 TOAHEX
2. 通过菜单 `文件 → 打开` 或拖拽存档文件到窗口
3. 选择 TOA_XXX 或 TOASYS 格式的存档文件
4. 编辑各项数据
5. 通过菜单 `文件 → 保存` 或 `Ctrl+S` 保存修改
> ⚠️ **注意**: 编辑存档前请务必备份原始文件。虽然工具会在保存时自动创建 `.bak` 备份，但仍建议手动保留一份原始存档。
当前程序还处于Release Preview阶段，需要备份好原始存档，以免造成数据损坏
